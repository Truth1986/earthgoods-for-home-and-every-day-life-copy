Vercel & Netlify have easy dashboards to add these.

## ❓ FAQ

**Q: Do I need to pay anything?**
A: No. Both Vercel & Netlify have free tiers. Only pay if you exceed limits (unlikely).

**Q: Can I use my own domain?**
A: Yes! Point DNS to their nameservers - takes 5 minutes.

**Q: How often is my site deployed?**
A: Auto-deploys on each git push to main (if set up).

**Q: How do I update my site?**
A: Push changes to GitHub - deployment happens automatically.

**Q: Is my site secure?**
A: Yes! SSL (HTTPS) is auto-enabled. Security headers are configured.

**Q: What if my site goes down?**
A: Vercel & Netlify have 99.99% uptime. They monitor 24/7.
