## Quick Start: Deploy in 5 Minutes

### Option 1: Deploy on Vercel (Recommended - Easiest)

1. **Sign up for free account**
   - Go to https://vercel.com/signup
   - Connect your GitHub account

2. **Import this project**
   - Click "Add New" → "Project"
   - Select this GitHub repository
   - Vercel will auto-detect Vite config

3. **Configure environment (if needed)**
   - Set `VITE_BASE44_APP_BASE_URL` if using Base44 API

4. **Deploy**
   - Click "Deploy" - it takes ~2 minutes
   - You'll get a live URL instantly

5. **Connect custom domain (optional)**
   - In Vercel Dashboard → Project Settings → Domains
   - Add your domain (e.g., earthgoods.com)
   - Update DNS records at your domain registrar

### Option 2: Deploy on Netlify

1. **Sign up for free**
   - Go to https://netlify.com/signup
   - Connect GitHub

2. **New site from Git**
   - Click "Add new site" → "Import an existing project"
   - Select this repo

3. **Build settings (auto-detected)**
   - Build command: `npm run build`
   - Publish directory: `dist`

4. **Deploy**
   - Click "Deploy site"
   - Live in ~2 minutes

### Option 3: Deploy on GitHub Pages (Free but Basic)

1. **Update vite.config.js**
   ```javascript
   export default defineConfig({
     base: '/',  // or '/repo-name' if not using custom domain
   })
   ```

2. **Add GitHub Actions workflow**
   - Create `.github/workflows/deploy.yml`
   - See below for template

## Environment Setup

### Required Environment Variables
```env
# For Base44 integration
VITE_BASE44_APP_BASE_URL=https://your-base44-instance.com

# For Stripe integration
VITE_STRIPE_PUBLIC_KEY=pk_live_...
```

Set these in your hosting platform's dashboard.

## DNS & Domain Setup

### Register a Domain
- **NameCheap**: https://www.namecheap.com
- **GoDaddy**: https://www.godaddy.com
- **Google Domains**: https://domains.google

### Point Domain to Vercel
1. Buy domain at registrar
2. In Vercel: Project Settings → Domains
3. Add domain
4. Follow Vercel's DNS instructions
5. Update DNS records (usually A records or CNAME)

### Submit to Search Engines
After deployment:

1. **Google Search Console**
   - Go to https://search.google.com/search-console
   - Add property with your domain
   - Submit sitemap (auto-generated by production app)

2. **Bing Webmaster Tools**
   - Go to https://www.bing.com/webmasters
   - Add site
   - Submit sitemap

3. **Generate Sitemap**
   - Your app should auto-generate at `/sitemap.xml`
   - If not, check production build
